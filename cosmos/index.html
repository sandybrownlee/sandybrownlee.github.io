COSMoS

Computing Science and Mathematics Skill Sharing

COSMoS aims to allow an exchange of skills with colleagues. We've set up COSMoS as a group which can meet monthly, be led by anyone in the department and used as a method of teaching each other via 1 hour open format sessions. This can range from advanced topics to sharing basic principles.

It's important to note this is not a research-based group - we already have these available. This is for the transfer of skills that you've picked up be you a researcher, lecturer, student or support staff. For further examples of possible subject areas, see below.
